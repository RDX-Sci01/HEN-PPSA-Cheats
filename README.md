# HEN PPSA Cheats Sync

This repository automatically syncs and filters cheat data from:

https://github.com/TeeKay87/HEN-Cheats-Collection

---

## 📁 What it includes

Only **PPSA entries** are kept from:

- mc4
- json
- shn

Both:
- `PPSA*` files
- `PPSA` lines in `.txt` files

---

## 🔄 Auto Sync

- Runs every 6 hours via GitHub Actions
- Can be triggered manually
- Always replaces old data with fresh upstream sync

---

## ⚠️ Note

This repo is fully automated.  
Do not edit `cheats/` manually (it will be overwritten).
